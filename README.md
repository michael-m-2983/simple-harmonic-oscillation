# simple-harmonic-oscillation

Single HTML file simulation/visualization of simple harmonic oscillation with a mass and spring.

## Usage

Visit <https://michael-m-2983.github.io/simple-harmonic-oscillation/> and use the mouse to control the position of the box. There are also sliders to control the spring constant and damping.
